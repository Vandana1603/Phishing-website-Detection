# Phishing Website Detection using Machine Learning

This project focuses on detecting whether a website is phishing or legitimate using Machine Learning techniques.
A classification model is trained on URL-based features to identify suspicious websites.

The goal of this project is to build a model that can classify a given website as:

-> Phishing

-> Legitimate

# Features

 Detect phishing websites using ML

 URL-based feature extraction

 Displays prediction with probability

 Interactive web app using Streamlit

 **Input :**

1.Manual feature input

2.Direct URL input

# Dataset Features Considered:

URL length

Number of dots

HTTPS usage

Subdomains

Special characters

# Algorithms Used

XGBoost Classifier -Primary Model

Random Forest -Baseline Model

# Technologies Used

Python

Pandas, NumPy

Scikit-learn

XGBoost

Streamlit

# Limitations

Uses limited URL-based features

May not detect highly advanced phishing attacks
